In this project, entitled "Project 1", two apps have been created.
App 1 is about Hare Krishna Dynamic Webpage
This app was for trial, to explore the Dynamic Webpage feature 
of Django based on URL input.
App 2 is about Monthly Challenge Dynamic Webpage
This app is taught in college, wherein at the end of the URL the user
inputs a number representing a month and gets Dynamic Webpage as 
output. The string text is converted to integer
